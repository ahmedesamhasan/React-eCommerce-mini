# Nova Store

A cleaned and improved React e-commerce demo built with Vite, Bootstrap, Redux Toolkit, and DummyJSON API.

## Highlights

- Product listing with search, filters, sorting, and pagination
- Product details view with image switching
- Persistent cart using localStorage
- Persistent wishlist using localStorage
- Reusable UI components for loading, empty, error, and page sections
- Small API service layer for cleaner data access
- Toast notifications for user feedback
- English and Arabic language toggle
- Static hosting friendly Vite setup

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
